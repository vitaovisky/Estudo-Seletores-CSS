Seletores CSS

O CSS possui 4 tipos de seletores

  1. Seletor por tag

  2. Seletor por Classe

  3. Seletor por Id

  4. Seletor Universal

  5. Seletor por atributo

Seletores por Tag:

É necessário declarar na arquivo CSS a tag que deseja estilizar. Essa seleção irá alterar todas as tags similares.
Tem menor peso entre os estilos. No exemplo foi selecionada a tag < p > e foi adicionada a cor darklategray a ela.

Seletor por Classe

É necessário declarar a classe na tag, utilizando class="nome-da-classe". E no arquivo CSS a classe é estilizada declarando .nome-da-classe

Possui maior peso no estilo do que a declaração pela tag.

A classe pode ser repetida em diversas tags, para a estilização de várias tags simultaneamente.

Seletor por Id

É necessário declarar a Id na tag, utilizando id="nome-do-identificador".
E no arquivo CSS o Identificador é estilizado declarando #nome-do-identificador.
Possui maior peso no estilo do que a declaração pela tag e pela classe.
O Id pode ser declarado somente em uma tag, não deve ser repetido no arquivo HTML.
Mesmo que uma mesma tag tenha Id e classe, a estilização do Id terá maior peso que a da classe. Independente da posição de ambos no arquivo CSS.

Seletor Universal

Declarado no arquivo CSS como *

Ira estilizar todas as tags da página. Normalmente será utilizado para atribuir um padrão único, podendo ser usado para o reset da página. Possui menos peso na estilização, até menos que a tag.

Neste documento foi utilizado para atribuir o tamanho da fonte em 20px e cor inicial como darkslateblue

Seletor de Atributo

Utiliza o atributo para seleção do estilo.

Declarado no arquivo CSS como [atributo]

Irá estilizar todas tags com o atributo selecionado.

É possivel alterar as formas de localizar as tags de várias formas.

  • [atributo]

    Irá localizar um atributo e estilizar todas tags com este atributo.

  • [atributo="valor-registrado-dentro-do-atributo"]

    Irá localizar um atributo que tenha o valor especifico mencionado.

  • [atributo~="valor-parcial-registrado-dentro-do-atributo"]

    Irá localizar um atributo que tenha o valor mencionado separado por espaço no atributo, independente da posição do valor.

  • [atributo~="valor-parcial-registrado-dentro-do-atributo"]

    Irá localizar um atributo que tenha o valor mencionado separado por espaço no atributo, independente da posição do valor.

  • [atributo|="valor-registrado-dentro-do-atributo"]

    Irá localizar o valor exato mencionado ou o valor seguido por um hífen.

  • [atributo|="valor-registrado-dentro-do-atributo"]

    Irá localizar o valor exato mencionado ou o valor seguido por um hífen.

  • [atributo^="prefixo-que-deseja-verificar-registrado-dentro-do-atributo"]

    Irá localizar um atributo que tenha o prefixo com o valor mencionado.

  • [atributo$="sufixo-que-deseja-verificar-registrado-dentro-do-atributo"]

    Irá localizar um atributo que tenha o sufixo com o valor mencionado.

  • [atributo*="valor-registrado-dentro-do-atributo"]

    Irá localizar um atributo que tenha o valor especifico mencionado, este valor mencionado pode estar registrado em qualquer parte do valor.
